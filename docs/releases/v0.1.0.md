# Seiza for Windows 0.1.0

This is the first public Windows preview of Seiza: a native WinUI astronomy
image viewer and plate-solving application powered by the shared Seiza Rust
core.

## Highlights

- Open FITS, XISF, PNG, JPEG, and TIFF images or complete image folders.
- Browse with a cached thumbnail rail and a GPU-backed pan/zoom viewport.
- Apply seven stretch models, ordered adjustment stages, RGB strategies,
  background extraction, histograms, and Richardson-Lucy deconvolution.
- Solve locally and display WCS grids, stars, deep-sky objects and contours,
  transients, solar-system objects, and motion vectors.
- Export full-resolution images with or without overlays and export FITS WCS
  sidecars.
- Copy and paste rendered images or complete processing adjustments.
- Download, verify, repair, and relocate the required astronomy catalogs.
- Install a self-contained application for all users with FITS and XISF file
  associations.

## Install

Download `seiza-0.1.0-windows-x86_64.msi` and compare it with the published
SHA-256 checksum. The installer requires Windows 11 24H2 or newer on x64 and
administrator approval because it installs for every user.

This preview is not yet code-signed, so Windows may display an
unknown-publisher warning. The MSI includes .NET 10, Windows App SDK/WinUI,
Win2D, and the Seiza Rust core; it does not download another runtime during
installation or first launch.

## Still in progress

Multiple document windows, 16-bit PNG/TIFF export, Explorer Preview/thumbnail
integration, ARM64, and production code signing remain planned work.
